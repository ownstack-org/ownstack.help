ownstack config

View

$ ownstack config [--app=NAME]                  # dump all vars
$ ownstack config:get KEY [--app=NAME]           # one var

Set & unset

$ ownstack config:set [--app=NAME] FOO=bar BAZ='multi word'
$ ownstack config:unset [--app=NAME] FOO BAZ
$ ownstack config:set --no-restart [--app=NAME] FOO=bar    # defer restart

Push from a file

$ ownstack config:push --app=NAME --file=heroku-config.env

Accepts the Heroku-style FOO='bar' shell-quoted format. Useful for migration; heroku config -s -a my-app > heroku-config.env emits exactly this.

Per-stack overrides

$ ownstack config:set --app=NAME --stack=STACK FOO=bar

Stack-level vars override app-level on that stack only. See Config vars.
